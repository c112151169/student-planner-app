# 校園生活提醒 APP（Student Planner）

本專案為一套使用 **Flutter** 開發之校園生活提醒應用程式，  
提供學生簡單直覺的方式來管理課業、考試與日常活動，  
並支援本地資料儲存與離線使用。

---

## 📌 專案功能

- ✅ 新增提醒事項（標題、日期、分類）
- 📂 事項分類管理（作業 / 考試 / 社團 / 其他）
- ✔ 完成 / 未完成 狀態切換
- 🔍 事項搜尋功能
- 🎯 篩選功能（全部 / 已完成 / 未完成）
- 📅 依日期或新增時間排序
- 📊 今日 / 七天內未完成事項統計
- 🗑 左滑刪除事項
- 🌙 深色模式切換（Dark Mode）
- 💾 SQLite 本地資料庫儲存（離線可用）

---

## 🏗 系統架構說明

本系統採用 **三層式架構設計**：

- **使用者介面層（UI Layer）**  
  使用 Flutter Widgets 建立操作介面，包含事項清單、新增頁面、搜尋與篩選功能。

- **應用邏輯層（Logic Layer）**  
  處理事項新增、刪除、完成狀態切換、搜尋、排序與統計邏輯。

- **資料儲存層（Data Layer）**  
  使用 SQLite 作為本地資料庫，透過 sqflite 套件進行資料存取。

---

## 🧭 操作流程簡述

1. 使用者開啟應用程式  
2. 系統從 SQLite 載入已儲存之事項資料  
3. 顯示事項清單與統計資訊  
4. 使用者可新增、搜尋、篩選、完成或刪除事項  
5. 系統即時更新資料庫並重新顯示清單

---

## 🛠 開發環境與技術

- **開發框架**：Flutter
- **程式語言**：Dart
- **資料庫**：SQLite
- **執行平台**：Windows Desktop
- **資料儲存套件**：sqflite / sqflite_common_ffi
